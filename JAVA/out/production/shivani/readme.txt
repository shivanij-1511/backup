----COMMANDS----
command to compile source code:
javac Filename.java

command to interpret code/bytecode:
java Filename

command to get all the methods:
javap java.util.Scanner

command to get all const constable
javap java.lang.Integer
javap java.lang.Byte
javap java.lang.Float
javap java.lang.Double
similarly for all datatypes

---IMPORTANT KEYPOINTS---
filename must be same as the class name if the class is of public type, otherwise filename and className can be different.
